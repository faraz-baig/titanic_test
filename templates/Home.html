Project Aim

The main purpose of this project is to predict whether a person has heart disease of not?
Input and output
Input: Person
Output: Prediction (Have Heart Disease / Does not have heart disease)
Representation of Input (Passenger)
Input (Person) is represented as the following four input attributes:
Input Attributes 01– GENDER
Possible Value 01 – Male
Possible Value 02 – Female
Input Attributes 02– FBS
Possible Value 01 – True
Possible Value 02 – False
Input Attributes 03– Exang
Possible Value 01 – yes
Possible Value 02 – no
Input Attributes 04– Slope
Possible Value 01 – Upsloping
Possible Value 02 – Flat
Possible Value 03 – Downsloping
Representation of Output
Output is Represented as a single Output Attributes
Output Attribute – Have Heart Disease
Possible Value 01 – Yes
Possible Value 02 – No
